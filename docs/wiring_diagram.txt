CONNECTIONS:
============

DS18B20 TEMPERATURE SENSOR:
---------------------------
DS18B20 Red    → Pi Pin 1  (3.3V)
DS18B20 Black  → Pi Pin 9  (GND)
DS18B20 Yellow → Pi Pin 7  (GPIO 4)

Note: 4.7kΩ pull-up resistor between Data and VCC (usually built into waterproof modules)

RELAY MODULE (12V):
-------------------
Relay VCC → 12V Adapter (+)
Relay GND → Pi Pin 14 (GND) + 12V Adapter (-)
Relay IN  → Pi Pin 11 (GPIO 17)

12V PUMP CONNECTION:
--------------------
12V Adapter (+) → Relay COM (Common)
Relay NO (Normally Open) → Pump (+)
Pump (-) → 12V Adapter (-)

SAFETY NOTES:
=============
- Use optically isolated relay module
- Ensure proper grounding
- Check all connections before powering on
- Use appropriate wire gauge for 12V/3A load
- Keep 12V circuit separate from Pi circuits
- Test with multimeter before connecting pump

TROUBLESHOOTING:
================
- If DS18B20 not detected: Check 1-Wire is enabled in /boot/config.txt
- If relay doesn't click: Check 12V power and GPIO 17 signal
- If pump doesn't run: Check 12V power supply and relay contacts
- Use `gpio readall` command to verify pin states
